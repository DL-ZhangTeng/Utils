
# Utils
包含一些常用工具类
[GitHub仓库地址](https://github.com/DL-ZhangTeng/Utils)
## 引入
### gradle
```groovy
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}

implementation 'com.github.DL-ZhangTeng:Utils:1.0.0'
```

## 混淆
-keep public class com.zhangteng.**.*{ *; }
